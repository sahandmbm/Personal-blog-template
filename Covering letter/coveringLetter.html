Covering letter

Overview

THE BACKEND OF THIS WEBSITE HAS SEVERAL SECTIONS, INEACH SECTION DIFFERENT TECHNIQUE AND TECHNOLOGY IS USED, THE HIGHLIGHT OF THE BACK END IS:
- LOGIN SESSION FOR THE BLOG OWNER WITH LOGOUT AND SESSION REMOVE SO THAT GOING BACK TO PREVIOUS PAGE IS BLOCKED
- DATA BASE CREATED AND DATA INSERTED, UPDATED OR CAN BE DELETED
- AUTOMATIC EMAIL SENDING
- LOADING BLOGS FOR THE VIEWRS FROM THE DATABASE
It is important to mention that all of technologies required for this project is used.

Creating the databse: (DATABASE DRIVEN PAGE)

The "DBcreate.php" file creates a database and has the needed attributes for the blogger.

Login session:(PAGES WITH ACCESS CONTROL)

The login session firstly checks the username and password, gives and error or will redirect the user to the setting page if the information is correct.
The default username and password are admin, admin.
If the user decide to logout from setting page, the session is destroyed and the user cannot go back in pages and will be redirected to the login page.
All of the following pages after this page is access controlled and the session is checked in every page.

Adding blog: (DATABASE DRIVEN PAGE, Back-end pages to key in data into the database)

The user can upload image, inser date and other information and save the data into the database from the Adding page.

Editing blog: (DATABASE DRIVEN PAGE, Back-end pages to key in data into the database)

The posted blogs are fetched from the database and the user can choose to edit the blog. The information of the selected blog is displayed in the form and the user can edit it and save. a global variable is used to pass the information about the selected blog.

Deleting blog: (DATABASE DRIVEN PAGE, Ajax, Back-end pages to key in data into the database)

Ajax is used in this part to pass the information to another PHP file so that the query in that file can delete the selected blog.

Display blog: (DATABASE DRIVEN PAGE)

In the front end section (BLOG PAGE FOR THE VIEWERS), all of the blogs are gathered from the database and shown. there is a character limit for showing the text of the blog and the user can click on showmore to view all of the blog.

Email sending: (Use of technologies not covered in the lectures)

In contact page of the frontend, the viewer can fill a form and send a message to the owner of the blog. This is done by sending automatic emails, one copy to the sender and one copy to the reciever.
